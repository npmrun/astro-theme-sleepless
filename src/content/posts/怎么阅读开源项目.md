## 思路

1. github 的 commit 历史记录

    例如[vue3 的源码](https://github.com/vuejs/core/commits/main)的提交，可以清晰的看到每一个提交的具体内容。

2. git log --reverse

    执行`git log --oneline --reverse > commits.txt`,会将所有的提交信息打印到 commits.txt 中，之后可以按顺序一个个 checkout 即可
